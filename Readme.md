# ViralCraft

AI-powered video intelligence platform that converts long-form YouTube videos into viral short-form content automatically.

## Features

- YouTube video ingestion
- AI transcript analysis
- Viral clip detection
- Shorts generation
- Auto captions
- YouTube publishing

## Tech Stack

Frontend:
- Next.js
- TypeScript

Backend:
- FastAPI

Database:
- PostgreSQL

Queue:
- Redis

Storage:
- AWS S3
